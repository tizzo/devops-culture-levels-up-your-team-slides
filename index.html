Culture

A set of shared mental assumptions that guide interpretation and action in an organization by defining appropriate behavior or various situations

  • from (Ravahi & Shultz, 2006 http://www.jstor.org/stable/20159775)
  • via http://www.slideshare.net/matthewskelton/what-does-devops-culture-feel-like

Zivtech's Cultural Assumptions

  • Everything should be repeatable
  • Develop locally, deploy for integration
  • If it matters, it is in version control
  • Everything gets reviewed
  • Open source whatever you can
  • Communicate (more on this in the sharing section)
